# 美食推荐插件

基于 AstrBot 框架的美食推荐插件，向机器人发送特定消息即可获得随机美食推荐及详细做法。

---

## 功能介绍

- **随机推荐**：发送指定消息，机器人随机推荐一道美食并提供详细做法
- **按餐类筛选**：支持早餐、午餐、晚餐、夜宵四种餐类的筛选推荐
- **菜谱查询**：根据菜名查询具体做法
- **菜谱总览**：查看已收录的全部菜品列表
- **自然语言触发**：无需严格命令格式，用口语化的方式即可触发

---

## 使用方法

### 安装

将 `astrbot_plugin_food_recommend` 文件夹放入 AstrBot 的插件目录：

```
AstrBot/data/plugins/astrbot_plugin_food_recommend/
```

然后在 AstrBot WebUI 的「插件管理」页面启用本插件。

### 触发方式

#### 方式一：自然语言（推荐）

直接在聊天中发送包含以下关键词的消息即可：

- `吃什么` / `吃啥` / `吃点啥` / `想吃啥`
- `推荐点吃的` / `推荐美食` / `推荐食物`
- `不知道吃什么` / `今天吃什么`
- `美食推荐`

**指定餐类示例：**
- `早餐吃什么`
- `午餐吃啥`
- `晚餐推荐点吃的`
- `夜宵吃什么`

#### 方式二：命令

使用 `/` 前缀的命令（支持命令别名）：

| 命令 | 别名 | 说明 |
|------|------|------|
| `/吃什么` | `/吃啥`, `/food`, `/今天吃什么`, `/推荐食物` | 随机推荐美食 |
| `/吃什么 早餐` | — | 指定餐类推荐（早餐/午餐/晚餐/夜宵） |
| `/怎么做 菜名` | `/howto`, `/食谱`, `/做法`, `/recipe` | 查询指定菜品的详细做法 |
| `/菜单` | `/menu`, `/全部菜谱`, `/美食列表` | 列出所有已收录菜品 |

## 自定义菜品

编辑 `main.py` 中的 `FOOD_DB` 列表，按照以下格式添加菜品：

```python
{
    "name": "菜品名称",
    "category": "早餐",       # 早餐/午餐/晚餐/夜宵
    "difficulty": "⭐ 简单",   # 难度标注
    "time": "30分钟",         # 耗时
    "desc": "简短描述",
    "ingredients": "食材列表",
    "steps": [
        "步骤1...",
        "步骤2...",
    ],
},
```

修改后在 WebUI 点击「重载插件」即可生效。
